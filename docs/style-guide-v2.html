Proposed System Rules

RoleColorUsed ForPreview
Action#73b6c9Send buttons, generate, create, primary CTAAction
Confirm#000000Ship, receive, approve, current tab, primary textConfirm
Warning#f4b22bPending, waiting, rush, attention needed, partialWarning
Urgent#fd3aa3Hot priority, fulfillment, urgent actions, overdue follow-upsUrgent
Active#73b6c9Production, receiving, shipped status, in-progress workActive
Success#4ddb88Complete, paid, approved, all clearSuccess
Error#ff324dOverdue, on hold, void, variance, errorsError
Neutral#000000Inactive, default, intake, normal priorityNeutral

Consistent Rules

  • All pills: borderRadius: 99px — no exceptions
  • All pills: font-size: 10-11px, font-weight: 600, padding: 3px 10-12px
  • All buttons: borderRadius: 8px, font-size: 13px, padding: 10px 24px
  • Dim backgrounds always pair with darker text of the same hue
  • No dark backgrounds with light text on pills (only on buttons)
  • No dots, icons, or decorations on status pills
