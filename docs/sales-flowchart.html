Fixes Applied

#1 Costing gate added to QUOTE alert
QUOTE alert now requires costing_summary.grossRev > 0. No alert fires until costing is complete — project stays in "ready for costing" state.
#2 Net 15/30 invoice send alert now fires
Net terms clients get a purple P2 "Send invoice" alert. Same email modal, just doesn't gate production. Clears when payment is recorded.
#3 "Upload proofs" replaces "Send proofs" when no proofs exist
When items have zero proof files, alert now says "Upload proofs" and navigates to the Art tab. No more emailing an empty portal.
#4 No-contacts popup warning
Clicking QUOTE or SEND with no email contacts shows a modal: "Add an email address — this project has no email contacts." Button navigates to the project.
